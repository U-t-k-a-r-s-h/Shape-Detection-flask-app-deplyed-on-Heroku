Architecture:This model is trained using the Convolutional Neural Network architecture. Since, the aim of the model is to classify between 4 different classes, that's why
we have used 4 output units at the end of the model.
Activation Function:The Activation function for the hidden units is "relu" and the activation function at the output layer is softmax because we are dealing with probabilities.
Loss Function:The loss used for this model is sparse categorical crossentropy loss function as the standard loss function.

Training Accuracy = 96.2% Training loss = 0.000024
